# Discord widget redirect
Cloudflare Worker to redirect users to your discord server using the widget API.

> [!IMPORTANT] 
> Make sure to enable the Server Widget by going to <kbd><kbd>Server Settings</kbd> â‡’ <kbd>Engagement</kbd></kbd>, checking <kbd>Enable Server Widget</kbd> and selecting an <kbd>Invite Channel</kbd>.

## Setting up a development environment

- Clone this repo
- `npm install`
- Run `npm run dev` to get the development server up
- Access it through http://localhost:8787/
